# USB Camera & Microscope Setup Guide

## Supported Devices

### USB Webcams
- **Recommended**: Logitech C920/C922, Microsoft LifeCam, Razer Kiyo
- **Minimum**: 720p resolution, 30 FPS
- **Features**: Autofocus, low-light correction

### USB Microscopes
- **Professional**: Dino-Lite AM4113ZT, AM4515ZT series
- **Budget**: Jiusion 40x1000x, Plugable USB-MICRO-250X
- **Features**: LED lights, adjustable focus, high resolution

## Setup Instructions

### 1. Hardware Setup
1. Connect USB camera/microscope to your computer
2. Position device 2-6 inches from plant surface
3. Ensure good lighting (use microscope LEDs or external lights)
4. Adjust focus for clear image

### 2. Browser Permissions
1. Click "Allow" when prompted for camera access
2. Select the correct USB device from the dropdown
3. Grant permissions if blocked by browser

### 3. Optimal Settings

#### For Webcams:
- **Resolution**: 1280x720 (HD)
- **FPS**: 30
- **Focus**: Auto
- **Lighting**: Bright, indirect light

#### For Microscopes:
- **Resolution**: 1920x1080 or higher
- **FPS**: 15 (for better detail)
- **Focus**: Manual
- **Magnification**: 50x-200x for leaf analysis
- **LED Brightness**: 70-80%

## Troubleshooting

### Camera Not Detected
- Check USB connection
- Try different USB port
- Restart browser
- Clear browser cache

### Poor Image Quality
- Clean camera lens
- Adjust lighting
- Check focus settings
- Ensure stable camera position

### Permission Issues
- Click lock icon in browser address bar
- Allow camera access for this site
- Restart browser after granting permissions

## Best Practices

### For Plant Analysis:
1. **Consistency**: Use same angle and distance for each analysis
2. **Lighting**: Maintain consistent lighting conditions
3. **Focus**: Ensure leaf details are clearly visible
4. **Timing**: Analyze at same time of day when possible

### For Continuous Monitoring:
1. **Stable Setup**: Mount camera securely
2. **Power**: Ensure stable power supply
3. **Network**: Maintain stable internet connection
4. **Position**: Avoid direct sunlight or glare
